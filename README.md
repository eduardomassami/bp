# 1 - Instalar o NestJS
```bash
$ npm i -g @nestjs/cli
```

# 2 - Instalar dependências
```bash
$ npm install
```


# 3 - Rodar a simulação
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# 4 - curl para a simulação
```
curl --request GET \
  --url http://localhost:8080/
```
